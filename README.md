# Beginner Python Exercises

A collection of beginner-friendly Python console programs.  
This repository documents my learning progress as I practice fundamental programming concepts such as variables, loops, conditionals, functions, user input handling, and simple algorithms.

Each script is independent and can be executed directly from the terminal.

## ğŸ› ï¸ Key Skills Demonstrated

- **Modular Programming:** Logic organized into functions (`def`) following the **DRY** (Don't Repeat Yourself) principle.
- **Error Handling:** Use of `try...except` blocks to manage common runtime errors (`ValueError`, `IndexError`) gracefully.
- **Data Structures:** Practical manipulation of `list`, `tuple`, and string methods.
- **File Handling & Portability:** Use of absolute paths (`os.path`) and cross-platform console clearing (`os` / `platform`).
- **User Interaction:** Menu-driven console interfaces for better usability.
